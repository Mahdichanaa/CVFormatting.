CVFormatting.

In this folder, you'll find three distinct documents. 

Word template file : this template file tries to replicata the Symolia CV template
JSON File : text template file, contains the keys/items that define what type of information will be filled (for example, skills, experience professionel, etc)


3 Python files : 
1/ Allows to open the word and replace the information in the word file with the information in the JSON File
2/ Allows to fill the json information
3/ Pipeline file that allows to do all three.
